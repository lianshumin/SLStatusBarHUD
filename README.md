# SLStatusBarHUD
状态栏指示器小框架

## 显示成功信息
```objc
[SLStatusBarHUD showSuccess:@"加载成功"];
```

## 显示失败信息
```objc
[SLStatusBarHUD showError:@"加载失败"];
```

## 显示加载信息
```objc
[SLStatusBarHUD showLoading:@"正在加载中..."];
```

## 隐藏
```objc
[SLStatusBarHUD hide];
```

## 显示文字
```objc
[SLStatusBarHUD showMessage:@"abc..."];
```
